# Week 4 Reflection

Reflect 4: Is AI a collaborator, tool, or plagiarist in storytelling?
Personally, I see AI as both a tool and a collaborator. It’s a tool because I can use it to generate ideas, drafts, or visuals whenever I need help getting started. At the same time, it feels like a collaborator because it can suggest things I wouldn’t have thought of on my own, which can push my ideas in new directions.
